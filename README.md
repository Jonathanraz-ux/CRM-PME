# CRM PME

Application CRM pour PME avec gestion des leads, clients, devis, historique visuel et mode sombre.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Renseignez `VITE_SUPABASE_URL` et `VITE_SUPABASE_ANON_KEY` dans [.env.local](.env.local) avec les identifiants de votre projet Supabase
3. Run the app:
   `npm run dev`
